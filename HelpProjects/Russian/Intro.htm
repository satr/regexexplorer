Введение

Продукт RegexExplorer предназначен для разработки шаблонов регулярных выражений.

Возможности

  • Проверка произвольной строки текста (в том числе многострочного) на соответствие шаблону. Это простой способ проверить правильность шаблона или соответствие ему конкретного текста
  • Выборка из произвольной строки текста (в том числе многострочного) подстрок, соответствующих шаблону. Режим наиболее полезен для поиска информации в лог-файлах. Возможна загрузка произвольных файлов данных
  • Проверка набора строк текста, на соответствие шаблону - каждая строка проверяется независимо от других. Этот режим полезен при разработки шаблонов проверки стандартных полей ввода, имеющих много вариантов допустимого содержимого - имен, почтовых и электронных адресов, телефонов и т.п.

Другие возможности

  • Получение коллекции символов, использованных в списке проверяемых строк. Полезно для разработки шаблона по уже имеющимся данным.
  • Ввод специальных символов (в том числе кодами UNICODE).
  • Ведение собственной коллекции шаблонов.
  • Сохранение истории разработки шаблонов.
  • Сохранение история загружаемых файлов.
  • Загрузка данных из буфера обмена или из файла со списком.
  • Сохранение списка проверяемых строк в файл.
