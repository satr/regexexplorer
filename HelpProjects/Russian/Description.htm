"Соответствует" - проверка на соответствие

Проверка текста (в том числе многострочного) на совместимость с шаблоном регулярного выражения

В поле "Проверяемый текст" ввести текст, который будет проверяться на соответствие шаблону

В поле "Шаблон регулярного выражения" ввести текст шаблона

Нажать кнопку "Проверить" или клавишу "Enter" (если курсор находится в поле ввода шаблона)

Если текст не соответствует шаблону регулярного выражения, в поле "Результат" будет отображен результат проверки: "Неверно")

Изменить текст шаблона

Нажать кнопку "Проверить" или клавишу "Enter" (если курсор находится в поле ввода шаблона)

Если текст соответствует шаблону регулярного выражения, в поле "Результат" будет отображен результат проверки: "ВЕРНО"

"Соответствия" - выборка коллекции соответствующих шаблону подстрок

Выделение из текста (в том числе многострочного) коллекции строк, соответствующих шаблону регулярного выражения

Порядок работы:

  1. В поле "Проверяемый текст" ввести текст, который будет проверяться на соответствие шаблону
  2. В поле "Шаблон регулярного выражения" ввести текст шаблона
  3. Нажать кнопку "Проверить" или клавишу "Enter" (если курсор находится в поле ввода шаблона)
  4. В поле ""Результат" будет отображен результат проверки: "N Совпадений" или "Нет соответствий"
  5. В списке "Соответствия" отобразится коллекция подстрок проверяемого текста, соответствующих шаблону регулярного выражения

в начало

"Проверить по списку" - проверка на соответствие списка строк

Проверка списка строк на совместимость с паттерном регулярного выражения

Порядок работы:

  1. В разделе "Список проверяемых строк" в поле ввода ввести текст, который будет проверяться на соответствие шаблону
  2. Нажать кнопку "Добавить" - текcтовая строка будет добавлена в список ниже, в колонку "Проверяемый текст".
  3. При необходимости ввести таким аналогично нужное количество проверяемых строк
  4. В поле "Шаблон регулярного выражения" ввести текст шаблона
  5. Нажать кнопку "Проверить" или клавишу "Enter" (если курсор находится в поле ввода шаблона)
  6. В поле "Результат" будет отображен результат проверки: "N соответствий (из M строк)" или "Нет соответствий"
  7. В списке "Проверяемый текст" в колонке "Результат" отобразится результат проверки коллекции проверяемых строк текста, на соответствие шаблону регулярного выражения: "ВЕРНО" или "Нет соответствий"
Ввод строки проверяемого текста

В разделе "Список проверяемых текстов" в поле ввода ввести текст, который будет проверяться на соответствие шаблону

Нажать кнопку "Добавить"

текcтовая строка будет добавлена в список ниже, в колонку "Проверяемый текст"

После добавления текста в список, поле ввода будет очищено

Для добавления еще одной строки в поле ввода ввести новый текст

Включение флага "Сохранить текст после добавления" позволит сохранить добавленный текст в поле ввода

Нажать кнопку "Добавить"

текcтовая строка так-же будет добавлена в список

Флаг "Сохранить текст после добавления" позволяет ввести серию строк, имеющих небольшие изменения (например варианты электронных адресов, отличающиеся количеством символов: "a@a.a", "a@a.aa", a@aa.aa").

Изменяя предыдущую строку...

...и добавляя ее в список,

можно быстро ввести большое количество похожих строк

Редактирование строки проверяемого текста

Навести курсор мыши на строку текста, который нужно отредактировать

Щелкнуть левой клавишей мыши - строка списка примет вид редактируемого текстового поля

Изменить текст


Нажать клавишу Enter - измененный текст сохраниться на месте прежнего в строке списка


Нажатие на клавишу ESC отменяет изменение строки

в начало

Удаление строки или нескольких строк проверяемого текста

Для удаления необходимо выделить нужную строку (или несколько строк, удерживая клавишу Shift или Ctrl)

Удаление производится нажатием клавиши "Delete" или пункт меню "Очистить"

Если ни одна строка не выделена - предлагается удалить все строки
